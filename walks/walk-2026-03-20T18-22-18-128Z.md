## 🛠️ Solution: Three-Layer Cycle Detection

### Layer 1: Oscillation Detection (NEW)

```javascript
// Detect A->B->A->B pattern in last 6 breadcrumbs
const recent = breadcrumbs.slice(-6);
if (recent[0] === recent[4] && recent[1] === recent[5] && recent[0] !== recent[1]) {
  console.log("🔄 SURGEON: OSCILLATION DETECTED! Breaking cycle with random turn");
  return { turn: true, angle: Math.floor(Math.random() * 360) };
}
```

**Triggers:** BEFORE returning to start of cycle  
**Escape:** Random turn (breaks pattern)  
**Prevents:** 4-8 wasted steps per oscillation

---

### Layer 2: Early Loop Detection (IMPROVED)

```javascript
// Only trigger for RECENT breadcrumbs (2-8 steps ago)
const recentBreadcrumbIndex = breadcrumbs.slice(-10).indexOf(currentLocation);
if (recentBreadcrumbIndex !== -1 && recentBreadcrumbIndex < 8) {
  console.log(`🔄 SURGEON: LOOP DETECTED (back to step -${10 - recentBreadcrumbIndex})! 180° escape`);
  return { turn: true, angle: 180 };
}
```

**Triggers:** When returning to location from 2-8 steps ago  
**Escape:** 180° snap-back  
**Prevents:** Continuing deeper into loop

---

### Layer 3: Entropy-Based Escape (LOWER THRESHOLD)

```javascript
// Surgeon uses MORE SENSITIVE thresholds than Explorer
const isLowEntropy = entropy.variance < 5 && entropy.avgVisits > 2;

if (isLowEntropy) {
  consecutiveTurns++;
  if (consecutiveTurns >= 2) {  // Surgeon: 2 ticks vs Explorer: 3
    return { turn: true, angle: Math.floor(Math.random() * 360) };
  }
}
```

| Algorithm | Variance Threshold | AvgVisits Threshold | Ticks to Escape |
|-----------|-------------------|---------------------|-----------------|
| **Explorer** | < 3 | > 3 | 3 |
| **Surgeon** | < 5 | > 2 | 2 |

**Surgeon escapes FASTER** because its goal is 1:1 efficiency.
