## 🧠 Why This Happens: The Heatmap Paradox

The heatmap algorithm is working **too well**:

```
Step 1: Walker explores street eastbound → all nodes become "hot"
Step 2: At end of street, all forward directions are "hot"
Step 3: Algorithm prefers "cold" areas → only option is 180° turn
Step 4: Walker goes back westbound → retracing visited nodes
Step 5: At other end, same problem → turns 180° again
Step 6: REPEATS forever (bouncing between two endpoints)
```

### The Core Problem

| Scenario | Algorithm Behavior | Result |
|----------|-------------------|--------|
| **Open grid** | Finds cold spots, explores new areas | ✅ Good |
| **Linear street** | All directions equally "hot" | ❌ Bounces back |
| **Dead end** | Systematic search finds exit | ✅ Good |
| **Isolated territory** | No cold spots anywhere | ❌ Retraces |

## 🛠️ Proposed Fix: Entropy-Based Exploration

### Concept

When the **entire visible territory is hot**, the algorithm should detect **low entropy** and switch to a different exploration mode.

### Implementation

```javascript
// In traversal.js - detect low entropy situation
function calculateEntropy(visitedUrls, currentLocation, orientation) {
  const scanAngles = [0, 60, -60, 120, -120, 180];
  
  const visitCounts = scanAngles.map(angle => {
    const testOrientation = normalizeAngle(orientation + angle);
    const testLocation = predictNextLocation(currentLocation, testOrientation);
    return visitedUrls.get(testLocation) || 0;
  });
  
  const avgVisits = visitCounts.reduce((a, b) => a + b, 0) / visitCounts.length;
  const variance = visitCounts.reduce((sum, v) => sum + Math.pow(v - avgVisits, 2), 0) / visitCounts.length;
  
  return { avgVisits, variance, visitCounts };
}

// In decide() function
const entropy = calculateEntropy(visitedUrls, currentLocation, orientation);

// If all directions are equally "hot" (low variance), use random exploration
if (entropy.variance < 2 && entropy.avgVisits > 5) {
  console.log("🎲 LOW ENTROPY detected - using random exploration");
  return { turn: true, angle: Math.floor(Math.random() * 360) };
}
```

### Expected Behavior

| Entropy State | Condition | Behavior |
|--------------|-----------|----------|
| **High entropy** | variance > 10 | Normal heatmap scoring |
| **Medium entropy** | 2 < variance < 10 | Reduced breadcrumb penalty |
| **Low entropy** | variance < 2, avgVisits > 5 | Random exploration mode |
