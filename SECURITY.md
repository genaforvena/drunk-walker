# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by opening a security advisory in the GitHub repository.

**Do not** open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | ✅                 |
| < latest| ❌                 |

Always use the latest version for the most secure experience.

## Security Considerations

### What Drunk Walker Does

- Runs entirely in your browser console
- Simulates keyboard input (Arrow Up, Arrow Left)
- Records Street View URLs you visit
- Stores data in browser memory only

### What Drunk Walker Does NOT Do

- Send data to external servers (unless you manually export)
- Access your Google account
- Modify Google Maps functionality
- Install any software on your system

### Best Practices

1. **Only run in trusted environments** - Browser console has full page access
2. **Review code before running** - All code is open source
3. **Don't export sensitive data** - Path data is local only
4. **Use responsibly** - Respect Google's Terms of Service

## Privacy

- No telemetry
- No analytics
- No data collection
- All data stays in your browser
