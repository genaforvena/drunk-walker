CORE FEATURES:
🔄 Auto-Unstuck: Turns left 30°-90° and steps forward immediately.
⌨️ Arrow Up Movement: Simulates Arrow Up key press for smooth forward movement.
🎚️ Adjustable Pace: Control walking speed from 0.5 to 5.0 seconds per step.
⏸ Smart Pause: Pauses when you drag to look around, resumes automatically.
📍 Path Recording: Automatic path recording with turn angles and location.
🧭 Self-Avoiding Walk: Turns left at visited locations to explore new areas.
📂 Restore Walks: Load and continue from saved walk JSON files.
🔒 Never Gets Stuck: Random bounded left turns guarantee eventual escape.
🛠️ Zero Install: Runs entirely in your browser console.
HOW TO RUN (Developer Console):
1. Open Google Maps Street View.
2. Press F12 (or Right-Click > Inspect) and go to the Console tab.
3. Paste (Ctrl+V or Cmd+V) and press Enter.
4. Type allow pasting if your browser asks, then paste again.
5. The control panel appears—click START to begin walking!
WHAT HAPPENS:
• Script presses Arrow Up every pace interval (step counter increments)
• Auto-unstuck turns left 30°-90° then immediately steps forward
• Self-avoiding turns left 20°-50° at visited locations
• Path recording saves URLs, locations, and cumulative turn angles
• Copy/Download buttons export your walk as JSON
• Restore Walk loads and continues from saved JSON files
Works on any browser. No extension required.
