📦 Browser Extension (Recommended)

Install once, use anytime. No copy-pasting needed!

🌐 Chrome / Edge / Brave ~40 KB • Download ZIP 🦊 Firefox ~40 KB • Download ZIP

🔧 Installation: Chrome / Edge / Brave

  1. Download the ZIP file using the button above
  2. Extract to a folder (remember where you put it)
  3. Open Extensions: Type chrome://extensions/ in address bar
  4. Enable Developer Mode: Toggle switch in top-right corner
  5. Load Unpacked: Click button → select your extracted folder
  6. Done! Extension icon 🤪 appears in toolbar

🔧 Installation: Firefox

  1. Download the Firefox ZIP (orange button) using the button above
  2. Extract to a folder
  3. Open Debugging: Type about:debugging#/runtime/this-firefox
  4. Load Temporary Add-on: Click button → select manifest.json from your extracted folder
  5. Done! Extension loads until Firefox restarts
⚠️ Firefox Note:
  • Make sure you downloaded the Firefox ZIP (orange button), NOT the Chrome ZIP
  • Temporary add-ons unload when browser closes. Reload from about:debugging after restart
  • If you get "Extension is invalid" error, delete the extracted folder, re-download the ZIP, and try again

🎮 How to Use

  1. Open Google Maps
  2. Enter Street View mode (click on any blue location)
  3. Click the Drunk Walker extension icon 🤪 in your toolbar
  4. Click START and watch it explore!
